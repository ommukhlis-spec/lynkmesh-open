# Changelog

All notable public-facing changes to LynkMesh will be documented in this file.

LynkMesh is currently an early alpha / research preview. Public APIs and
generated artifacts may change. This project is not production-ready.

## [Unreleased]

### Added

- Five-minute public onboarding guide (`docs/try_in_5_minutes.md`).

- Public support matrix (`docs/support_matrix.md`).

- Agent review workflow documentation (`docs/agent_review_workflow.md`).
- Public evidence pack visibility for before/after evaluation baseline.
- Root Apache License 2.0 file (`LICENSE`) for public release readiness.

### Notes

- LynkMesh is not production-ready.
- Current evidence is an evaluation baseline, not benchmark proof.
- PHP has partial semantic support; other languages may fall back to
  structural-only analysis.
- Generated artifacts are deterministic static-analysis / graph-context
  artifacts and do not contain embedded LLM inference.
- Public APIs and CLI contracts are still evolving.
